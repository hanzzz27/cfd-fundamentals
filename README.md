## 🎯 Purpose

CFD is complex — but it doesn't have to be confusing.
This repository breaks down CFD concepts into simple, readable notes
based on my academic experience and undergraduate thesis research.

If you're a student, engineer, or just curious about fluid simulation,
this is for you.
